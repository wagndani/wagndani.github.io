Let's review.

    In this tutorial we:
  • Learned that the Recreation Information Database is a wealth of information available to anyone who enjoys the public recreational opportunities provided by Federal and State Agencies
  • Learned what an API is, how to use it, and what benefits API offer
  • Learned how to generate a table of governmental agencies that participate in the RIDB
  • Learned how to query recreational areas by proximity to a place (town / city), and how to generate web content for those areas
  • Learned how to nest a second XMLHttpRequest callback to bring in imagery for the recreation areas. (Don't get too crazy with this, or callback hell will ensue)
  • Learned how to query reservation data for a specific location and time period
  • Learned how to work with the reservation data to filter and reduce it to something a bit more manageable
  • Learned how to take our summary objects and make interactive charts out of them using the Google Charts API

I sincerely hope you stuck it out this far and learned something, and more importantly, will put your knowledge to good use. I certainly learned a lot from this project, and plan to build another site where the user can work with these types of queries live.
