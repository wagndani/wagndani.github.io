Georgia On My Mind

North Georgia specifically. One of my favorite places to go within a few hours drive of home is the mountains and hills of northern Georgia in the fall. There is a popular pumpkin patch just outside of Dawsonville, GA we have made a tradition to visit annually. So, lets see what we can find nearby...

I'll start by entering Dawonsonville, GA into Google Maps. If I take a look at the pane that pops up on the left side of the page, and the address bar, I have information for the zip code, latitude and longitude for Dawsonville, GA. Now, looking back at the documentation for the RIDB under "Get All Recreation Areas" we find a listing of available "URL Parameters". We can use these parameters to filter through all the records in the RIDB to those which are specific to our needs. If I were interested in retriving records near a specific location, I can see that the "latitude", "longitude", and "radius" parameters will fit the purpose.

I'll start by creating an object populated with the data I found on Google Maps for Dawsonville, and a variable to hold the distance (miles) I am willing to search: 

        var Dawsonville = {
            latitude:34.43018,
            longitude:-84.12900,
            zipCode:30534,
            state:"Georgia",
            stateAbbrev:"GA"
        }

        var distance = 50;          /*50 miles being the max distance I can travel without complaints from the back seat*/

Now, modify the request from the organization pull from the previous page, using the latitude, longitude, and radius parameters by appending them into the URL with the ampersand (&):

        req.open('GET', 'https://ridb.recreation.gov/api/v1/recareas/?latitude=' + Dawsonville.latitude + "&longitude=" + Dawsonville.longitude + "&radius=" + distance + "&" + MyapiKey , true);

Looking through the return, You may notice a few attributes of particular interest: RecAreaName:, RecAreDescription:, RecAreaDirections:, keywords:, and RecAreaPhone: all look to be good resource information. Lets do something with it.

Adding the information to a website

We have seen how to create a GET request and how to store the response in a variable, let's review: 


/*Create an object of our target location*/
    var Dawsonville = {
        latitude:34.43018,
        longitude:-84.12900,
        zipCode:30534,
        state:"Georgia",
        stateAbbrev:"GA"
        }

/*Create a variable to store a specific distance from our target location*/
    var distance = 50;

/*Create the request*/
    var recAreaRequest = new XMLHttpRequest();

/*Define the query*/
    recAreaRequest.open('GET', 'https://ridb.recreation.gov/api/v1/recareas/?latitude=' + Dawsonville.latitude + "&longitude=" + Dawsonville.longitude + "&radius=" + distance + "&" + MyapiKey , true);

/*add an event listener and output object information to the console window*/
    recAreaRequest.addEventListener('load', function(){
        if (recAreaRequest.status >= 200 && recAreaRequest.status < 400){
            var dataCollected = JSON.parse(recAreaRequest.responseText);
            var targetObjects = dataCollected.RECDATA;
            console.log(targetObjects);
        }

        else {
            console.log("Error in network request: " + recAreaRequest.statusText);
        }
    ;}

    recAreaRequest.send(null);

This will output the object array of recreation areas to the console.

Now, let's modify the code to output this data to the browser, but before we do, we need to define a few more concepts.

var _someName = document.createElement("HTMLelement"); This will create a new HTML element of the defined type (ie. ("div") will create a <div>) and assign it to the variable _someName.

The text within that new element can be created by calling _someName.textContent = "Here is some text"; or if the JSON parsed string contains HTML tags within its body, then _someName.innerHTML = "<strong>This text will be bold </strong>"

To assign an elements text with the contents of an object instead of hardcoded strings, simply use the dot notation followed by the property name to access the description. For example using the Dawsonville object above the assignment _someName.textContent = Dawsonville.State will assign the string Georgia to the text content of _someName.

Now that we have created an element with something to display, we add it to the HTML document by appending the new element to some existing HTML element. You can append the new element into the <body> by using document.body.appendChild(_someName);. Or if you have a specific place you need the element to be placed and you have another tag with a unique id you can use document.getElementByID('elementID').appendChild(_someName);. For example, you may have a tag with a id as such: <div id='put_someNameHere> and you can access it by calling document.getElementById('put_someNameHere').appendChild(_someName);

Let's add one final building block before we do anything amazing. Let's talk about arrayName.forEach(); JavaScript contains a built in method for array called forEach();. Now we are finally ready to do something productive with our information we recieved from RIDB. Going back to our Object we created from our JSON return from the RIDB response, we see that we are dealing with an array of objects, and each object contains keys (property attributes) for the area name, description, directions, and phone number. Let's use this information to autogenerate a webpage for rec areas within 50 miles of Dawsonville, GA.

    var Dawsonville = {
        latitude:34.43018,
        longitude:-84.12900,
        zipCode:30534,
        state:"Georgia",
        stateAbbrev:"GA"
    }

var distance = 50;


    var req = new XMLHttpRequest();

        req.open('GET', 'https://ridb.recreation.gov/api/v1/recareas/?latitude=' + Dawsonville.latitude + "&longitude=" + Dawsonville.longitude + "&radius=" + distance + "&" + MyapiKey , true);
                req.addEventListener('load', function(){
                    if (req.status >= 200 && req.status < 400){
                        var dataCollected = JSON.parse(req.responseText);
                        var targetObjects = dataCollected.RECDATA;
                        console.log(targetObjects);

                  /*The following code with create a div for each object in the retured array.
                  Each div will contain a h3 header for the area name, a block for the rec area
                  description, a block for directions to the rec area, and a contact phone number.
                  The div for each area (oject) will then be added to the HTML body*/

                  targetObjects.forEach(function(element){
                        var newRecOpp = document.createElement("div");

                        var newRecOppHeader = document.createElement("h3");
                            newRecOppHeader.textContent = element.RecAreaName;
                            newRecOpp.appendChild(newRecOppHeader);
                            newRecOpp.style.border = 'solid';

                        var aboutArea = document.createElement("h4");
                            aboutArea.textContent = "About Area:";
                            var aboutAreaText = document.createElement("p");
                            aboutAreaText.innerHTML = element.RecAreaDescription;
                            newRecOpp.appendChild(aboutArea);
                            newRecOpp.appendChild(aboutAreaText);


                        var directions = document.createElement("h4");
                            directions.textContent = "Directions:";
                            var directionsText = document.createElement("p");
                            directionsText.textContent = element.RecAreaDirections;
                            newRecOpp.appendChild(directions);
                            newRecOpp.appendChild(directionsText);


                        var contactInfo = document.createElement("h4");
                            contactInfo.textContent = "Contact:";
                            var contactText = document.createElement("p");
                            contactText.textContent = element.RecAreaPhone;
                            newRecOpp.appendChild(contactInfo);
                            newRecOpp.appendChild(contactText);


                        document.body.appendChild(newRecOpp);
});
                    }

                    else {
                        console.log("Error in network request: " + req.statusText);
                    }

                ;})

        req.send(null);

Allatoona Lake

About Area:

Allatoona Lake is one of the most frequently visited U.S. Army Corps of Engineers lakes in the nation. Located only 30 miles from Atlanta, more than 6 million visitors each year enjoy picnicking, camping, hunting, fishing, swimming and other water sports, and observing the abundant wildlife along the shores of this beautiful lake. Lake Sidney Lanier

About Area:

Nestled in the foothills of the Georgia Blue Ridge Mountains lies Lake Sidney Lanier, one of America?s favorite lakes. Over 7.5 million people a year choose to visit Lanier. Seventy-six recreational areas have been developed around the lake under the direction of the Corps of Engineers. These include 46 parks operated directly by the Corps, 10 marinas, and Lake Lanier Islands. The remaining areas are leased to county and city governments or other organizations such as the Boy Scouts. With over 692 miles of shoreline, the lake is well known for its aqua-blue colored water, spectacular scenery and variety of recreational activities. Constructed by the U.S. Army Corps of Engineers in the 1950?s, Lake Lanier is a multi-purpose lake that provides for flood protection, power production, water supply, navigation, recreation and fish and wildlife management. Lake Lanier is one of 464 lakes in 43 states constructed and operated by the U. S. Army Corps of Engineers. It has won the best operated lake of the year award in 1990, 1997 and 2002.
